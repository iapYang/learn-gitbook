# 配置

在书籍目录下的book.json

```json
{
    "plugins": [
        "jsfiddle",
        "expandable-chapters"
    ],
    "pluginsConfig": {
        "jsfiddle": {
            "type": "script",
            "tabs": [
                "result",
                "js",
                "css",
                "html"
            ],
            "height": "500",
            "width": "500",
            "fontColor": "00FF00"
        },
        "expandable-chapters":{}
    }
}
```

所有个性化的配置都在此处配置，大部分配置和插件的使用相关，具体的留到插件 部分详谈。
